# 🚀 AI Habit Tracker

AI-powered habit tracking web app built with the MERN stack.  
Track daily habits, visualize streaks with GitHub-style heatmaps, and get personalized AI insights using Google Generative AI.


## ✨ Features

- 📅 Track daily habits and streaks
- 🔥 GitHub-style activity heatmaps
- 🤖 AI-generated weekly reports & suggestions
- 📊 Analytics dashboard with charts
- 🎯 Habit recovery plans for broken streaks
- 🌙 Light/Dark mode support
- 🔐 JWT Authentication
- 📱 Responsive UI with Tailwind CSS


## 🛠️ Tech Stack

**Frontend:** React, Vite, Tailwind CSS, Recharts  
**Backend:** Node.js, Express.js  
**Database:** MongoDB  
**AI Integration:** Google Generative AI  
**Authentication:** JWT, bcryptjs


## 🚀 Getting Started

```bash
git clone https://github.com/varsha-1428/AIHabitTracker.git
cd AIHabitTracker
```

### Backend Setup

```bash
cd backend
npm install
npm run dev
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
